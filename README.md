# SkillInspect Web

Paste a public GitHub Agent Skill URL and receive a static pre-install report covering runtime commands, credential names, network hosts, file writes, external side effects, and traceable safety findings.

The browser reads public GitHub files directly. The site does not execute Skill code, store repository contents, or require an account.

## Development

```bash
npm install
npm test
npm run dev
```

The full local CLI remains available as [`skillinspect`](https://github.com/starinzlob/skillinspect).
